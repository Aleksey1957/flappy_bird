# Flappy Bird (Python + Pygame)

Это упрощённая и улучшенная версия игры **Flappy Bird**, написанная на
Python с использованием библиотеки **Pygame**.

В игре:

-   есть **таймер**, который показывает, сколько секунд игрок
    продержался\
-   каждые **10 секунд** игра автоматически ускоряется\
-   ускоряется скорость труб и сила гравитации\
-   управление осуществляется кликом мыши или клавишей **SPACE**

Проект использует ассеты, которые должны находиться в следующих папках:

    Flappy Bird Assets/
        Background/
        Player/StyleBird2/
        Tiles/Style 1/

## Начало работы
> RUS: Для запуска проекта достаточно запустить файл "main.py", используя Python версии 3.13.

После запуска:

-   нажмите **SPACE** или кликните мышью, чтобы взмахнуть крыльями\
-   избегайте труб\
-   после смерти нажмите **SPACE** или кликните, чтобы начать заново

## ⚙️ Механика ускорения

Каждые **10 секунд**:

-   скорость труб увеличивается на `+1`
-   гравитация увеличивается на `+0.2`
